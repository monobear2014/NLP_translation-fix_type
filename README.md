# NLP Translation & Text Correction
.

A simple Natural Language Processing (NLP) application built with Streamlit for language translation and text correction.

## Features

* Detect input language automatically.
* Translate text between multiple languages.
* Correct spelling and grammar mistakes.
* User-friendly web interface powered by Streamlit.
* Fast and interactive experience.

## Technologies Used

* Python
* Streamlit
* NLP Libraries
* Machine Translation Models

## Project Overview

This project demonstrates the application of Natural Language Processing (NLP) techniques to translate text and improve language quality through spelling and grammar correction. Streamlit is used to provide an intuitive and interactive web interface for users.
